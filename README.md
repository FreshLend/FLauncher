# FLauncher
Лаунчер для [VoxelCore](https://github.com/MihailRis/VoxelEngine-Cpp), написанный на Python.
Который старается быть похожим на TLauncher.

Особенности:
- Кроссплатформенность
- Работа в оффлайн режиме
- Discord Rich Presence
- Интеграция других репозиториев
- Похож на TLauncher

В планах:
- Страница модов

В далёких планах:
- FLauncher-Neo - дизайн лаунчера как Prism/MultiMC

## Установка библиотек
> pip install -r requirements.txt

## Сборка
> pip install pyinstaller  
> pyinstaller build.spec
