# FLauncher
FLauncher — Лаунчер для [Voxel Core](https://github.com/MihailRis/VoxelEngine-Cpp), написанный на Python.
Который старается быть похожим на TLauncher.

Особенности:
> Работа в оффлайн режиме
> Discord Rich Presence
> Похож на TLauncher

В планах:
> Страница модов
> Настройки
